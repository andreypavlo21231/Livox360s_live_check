# Livox360s_live_check
First of all - you need to set your network adapter settings ip to 192.168.1.5, gateway 192.168.1.1, and mask 255.255.255.0 and setup venv using python -m venv venv
then activate venv 
install req.txt by using "pip install -r req.txt"
then - use gather, by default it gathers 15secs of points, and writes to cloud.pcd file. 
Then - use visualize.py, for see result
enjoy :)
